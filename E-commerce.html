About The Project

This is a fictional design of an e-commerce store Fiuge, which delivers online groceries and some household services to their customer. The sole focus of this e-commerce project was to improve the customer experience and increase sales. Every design decision in this project is properly backed up by deep research on some top e-commerce brands and tries to pick out some of the best practices to increase sales for the business.

Categories page

Users can open the store and filter out any category product they want

Product Page

This page shows the description and features of any single product

Cart

All the products which are added to the cart will show here along with total amount customer has to pay.

Checkout

This checkout page shows all the essential information a user needs to know before finalising the order and completing the payment moreover, it asks essential details required for a successful delivery and also option for multiple payment modes
